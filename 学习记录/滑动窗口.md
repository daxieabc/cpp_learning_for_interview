1、什么类型的题目适合滑动窗口？  
连续子数组或子字符串相关的问题最适合
| 特征 | 适合滑动窗口 | 不适合滑动窗口 (建议其他方法) |
| :--- | :--- | :--- |
| 连续性 | 必须是连续的子数组/子串 | 非连续子序列 (DP/贪心) |
| 指针移动 | 左右指针均单向向右 (单调) | 左指针需回退或乱跳 |
| 目标 | 最长/最短/固定长度/特定统计 | 全局组合/乱序匹配 |
| 状态更新 | 进出元素可  $ O(1) $  更新状态 | 窗口内统计复杂 (如需排序/中位数) |
| 典型题型 | 最小覆盖子串、无重复字符最长子串 | 最长递增子序列、0-1背包 |
